Работа с базами данных

В этом разделе представлены результаты лабораторной работы №3 – создание и работа с базой данных в Microsoft Access.

Структура базы данных

Была создана база данных для учёта записи пациентов к врачу. Основные таблицы:

  • Пациенты - содержит информацию о пациентах (ID, ФИО, дата рождения, телефон)
  • Врачи - содержит информацию о врачах (ID, ФИО, специализация, кабинет)
  • Записи - содержит информацию о записях (ID пациента, ID врача, дата приема, время)

Схема данных

Схема связей между таблицами (скриншот из Access)

Пример SQL-запроса

SELECT Пациенты.ФИО, Врачи.ФИО, Врачи.Специализация, Записи.Дата_приема
FROM (Записи
INNER JOIN Пациенты ON Записи.ID_пациента = Пациенты.ID)
INNER JOIN Врачи ON Записи.ID_врача = Врачи.ID
WHERE Записи.Дата_приема BETWEEN #01.03.2026# AND #31.03.2026#
ORDER BY Записи.Дата_приема ASC;
